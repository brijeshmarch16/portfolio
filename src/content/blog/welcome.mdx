A minimal, pixel-perfect dev portfolio, and blog to showcase my work as a Frontend Engineer.

### Tech Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- retroui
- Fumadocs

### Features
- Clean & modern design
- Light & Dark theme support
- SEO optimization: JSON-LD schema, sitemap, robots
- Installable PWA

### Blog
- Built with Fumadocs
- Access markdown by adding a new file in the `src/content/blog` directory
- Syntax highlighting for code blocks
- Dynamic OG images for each blog post
